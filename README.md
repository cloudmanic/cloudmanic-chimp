# Cloudmanic Chimp

A Codeigniter package for interfacing with Mailchimp

## Requirements

1. PHP 5.1+
2. CodeIgniter 2.0.0+
3. CURL

## Development

This is still under heavy development. Do with it as you wish. Not very complete yet
